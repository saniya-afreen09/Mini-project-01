#Digital Attendance System SDG-4**

This project is an Arduino-based Digital Attendance Monitoring System designed to record and calculate student attendance using a 4x4 keypad and a 16x2 I2C LCD display.
The system allows the user to enter the total number of students, input roll numbers, mark each student as Present or Absent, and automatically calculate the overall attendance percentage.
It provides a simple, low-cost alternative to manual attendance tracking and demonstrates the use of embedded systems for real-time data processing and display.
